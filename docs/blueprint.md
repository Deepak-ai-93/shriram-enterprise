# **App Name**: PrintPro Showcase

## Core Features:

- Product Display: Showcase the T-Shirt printing machines with images and key specifications.
- Responsive Layout: Ensure the website is fully responsive across all devices.
- Contact Information: Display company contact details in the footer.
- Get Quote CTA: Prominent 'Get Quote' button for easy inquiries.
- Image Curation: Leverage a tool to select appealing placeholder images for machines.

## Style Guidelines:

- Primary color: Light Orange (#FFB74D) for buttons, highlights, and headers, conveying a modern and professional feel.
- Background color: White (#FFFFFF) to maintain a clean, modern aesthetic.
- Accent color: A slightly darker shade of orange (#F57C00) will be used for hover states, providing subtle visual feedback.
- Font: 'Inter', a sans-serif font, for a clean, contemporary feel.
- Use simple, professional icons (Lucide SVGs or similar) next to key features to enhance readability and engagement.
- Utilize rounded corners and subtle shadows to create a premium, modern look. Ensure a clean and uncluttered layout.
- Subtle fade-in animations using GSAP when scrolling to reveal content, creating a smooth user experience.
